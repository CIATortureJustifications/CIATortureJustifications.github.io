Black Site Locations & Detainee List

(Please note exact black site locations are classified, the map indicates the countries where sites were located but not exact location).

A black site is a location that is off the records that a detainee is taken to to be imprisoned and subjected to EITs Enhanced Interrogation Techniques
 , in other words, be torture. This is because at these sites no laws techically apply. Only a select amount of people knew the existance and locations of these sites.

All black sites were given a code name relating to a colour, for example, Thailand’s site was called ‘Detention Site Green’. The pin colours on the interactive map relates to the site code name colour. The full list of sites names are:

  1. ‘Green’
  2. ‘Cobalt’
  3. ‘Blue’
  4. ‘Gray’
  5. ‘Black’
  6. ‘Indigo’
  7. ‘Maroon’
  8. ‘Orange’
